3 ways to link up CSS to HTML

Inline: styles are written like attributes to html tags. Not recommended because no styles can be reused and these styles trump all (to be explained later).

Embedded/Internal: Written at the bottom of the html file. Also not recommended because the styles can only be used on that html file.

External: written in a separate file and linked. RECOMMENDED! Easiest to maintain.
