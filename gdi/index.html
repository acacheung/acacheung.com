What is HTML?

Stands for Hypertext Markup Language

Invented by Tim Berners-Lee in 1991 to share scientific papers

The structure: Semantics, content, data



Tags

HTML tags usually come in pairs (start tag and end tag / open tag and close tag)


Except for ones that cannot be contain other elements (ex: <br/>, <img/>)


Opening tag: <p> left-angle bracket, character, right-angle bracket

Closing tag: </p> left-angle bracket, forward slash, character, right-angle bracket

Tags

Tells the browser about your content


<html> describes the web page, meta info

<body> visible page content

<header> header content: logo, nav, search, login

<footer> footer content: related links, copyright

<div> defines a division or a section, a container/wrapper

<nav> navigational content: links to other pages or parts within a page

<aside> content related to its surrounding content, sidebar

<section> defines a section

<article> a self-contained piece of content. Ex: forum post, a magazine/newspaper article, a blog entry

More Tags

<p> paragraph text

<h1> heading

<h2> heading less important than h1

<span> groups inline-elements together

<ul> unordered list

<ol> ordered list

<li> list item

<a> link, a stands for anchor

<em> and <strong> are used for emphasis, but DO NOT tie styles to your html structure. Is what you are highlighting semantically more important or do you just want to bold/italicize for appearance?

Attributes

HTML elements can have attributes

Attributes are always specified in the start tag

<element name = 'value'>

<a href = 'http://dockyard.com/'> Link to Homepage </a>

Common attributes: class, id, href (stands for hypertext reference), src

Relative vs. absolute paths

Used for links and images

Relative paths are linked relative to where the file is. Links with no path info are found in the same directory. Subdirectories are listed without preceding slashes.

Absolute paths are linked explicitly. Refer to a specific location of a file, including the domain. Must use absolute paths when linking externally.

Images

Have three components: the image element tag, the src attribute and the alt attribute (to be W3C compliant)

Images are self-closing elements and DO NOT need a closing tag. Images cannot contain any other elements.

<img src ="http://girldevelopit.com/assets/pink-logo.png" alt = "Girl Develop It Logo"/>
